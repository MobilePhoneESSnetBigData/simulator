Detailed Description

This is actually a container for all the agents used for simulation. An agent could be an object of one the the derived classes of Agent. The Agents are kept in an unordered_multimap as pairs <string, Agent*> where the first element of the pair is the name of the concrete agent (a person, a mobile device, an antenna etc) and the second element is a pointer to the actual object (agent).
