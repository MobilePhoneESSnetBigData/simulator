\subsection{Detailed Description}
This is the map where the simulation takes place. It could be a simple rectangle or any kind of geometry object(s) read from a wkt file. The map has a boundary that is implemented as a Geometry object, a factory object of Geometry\+Factory type used to create other objects. All geometric and geographic features of the simulator uses the G\+E\+OS library. G\+E\+OS is an open source C++ library which is just a port to C++ of the well know Java Topology Suite.
